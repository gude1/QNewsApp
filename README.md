## Features

- Signup and login screen
- Bootsplash screen;
- News List screen
- Refresh and load more/ lazy loading functionality
- Profile screen

## Installation

How to Install & test the React Native App

```bash
git clone
cd into folder
yarn or npm install
npx react-native run-android
```

**Server:** Hacker news endpoint
